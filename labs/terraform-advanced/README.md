# Terraform Advanced

## What This Lab Covers

Building on terraform-basics/, this lab covers production-grade Terraform patterns used in real engineering teams.

- Modules — reusable Terraform components
- Workspaces — managing multiple environments (dev, staging, prod)
- Remote state with locking
- Complex AWS architectures — multi-tier applications, auto-scaling, load balancers
- Terraform with CI/CD pipelines
