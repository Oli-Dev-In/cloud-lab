# CI/CD — GitHub Actions

## Why CI/CD

CI/CD stands for Continuous Integration and Continuous Deployment. It is the practice of automatically testing, validating, and deploying code every time a change is pushed to a repository. Instead of manually deploying infrastructure or applications, a CI/CD pipeline does it automatically, consistently, and safely.

GitHub Actions is the CI/CD tool built directly into GitHub. It is free for public repositories and is one of the most widely used CI/CD tools in the industry. It appears in the majority of junior cloud/DevOps job postings.
