# Python Scripts

## Why Python

Python is the standard scripting language for Cloud and DevOps automation. It appears in 85% of junior cloud/DevOps job postings as a required skill — not a nice-to-have. In this context, Python is not used to build applications — it is used to automate infrastructure tasks, interact with AWS APIs, process data, and eliminate manual repetitive work.

## What This Lab Covers

- Python fundamentals for automation (not web development)
- File and system manipulation with Python
- Calling external APIs with Python
- boto3 — the Python library for interacting with AWS programmatically
- Automation scripts for AWS resource management
