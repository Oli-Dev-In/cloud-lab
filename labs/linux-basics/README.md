# Linux Basics

## Why Linux

Linux is the operating system that runs the vast majority of cloud infrastructure worldwide. Every AWS server, every Docker container, every Kubernetes node runs Linux. A Cloud/DevOps engineer who is not comfortable in a Linux terminal cannot do their job.

This lab documents my progression from zero to confident Linux CLI usage. Every command here has been typed manually, understood, and documented in my own words.

## What This Lab Covers

- Filesystem navigation and structure
- File and directory management
- Permissions and ownership
- Process management
- Networking from the terminal
- Bash scripting for automation
- Package management
- System monitoring
