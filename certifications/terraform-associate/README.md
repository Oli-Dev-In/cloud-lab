# Terraform Associate (003)

## What is this certification

Terraform is the industry-standard tool for Infrastructure as Code (IaC). Instead of clicking around in the AWS console to create servers and networks, you write code that describes your infrastructure — and Terraform builds it automatically, consistently, and repeatably.

The Terraform Associate certification validates that you can write, plan, and apply Terraform configurations in a real professional environment. Every infrastructure deployment in this lab is done via Terraform — never manually.
