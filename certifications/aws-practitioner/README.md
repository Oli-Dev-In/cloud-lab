# AWS Cloud Practitioner (CLF-C02)

## What is this certification

The AWS Cloud Practitioner is the entry-level AWS certification. It validates a foundational understanding of AWS Cloud concepts, services, security, and pricing. It is not a technical deep-dive — it proves you understand what AWS is, how it works at a high level, and why businesses use it.

This is the first step before the Solutions Architect Associate.
