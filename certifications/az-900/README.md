# AZ-900 Azure Fundamentals

## What is this certification

AZ-900 is Microsoft's entry-level Azure certification. It covers core Azure concepts, services, security, privacy, and pricing. It is the Azure equivalent of the AWS Cloud Practitioner.

The goal of this certification in this learning plan is not to become an Azure specialist — it is to ensure no job opportunity is blocked by a lack of Azure knowledge. A significant portion of the European cloud market runs on Azure (Cloudfide, Medius, Orange Services, VINCI). Being able to say "I understand both AWS and Azure" opens more doors.
