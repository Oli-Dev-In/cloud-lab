## Learned
- du: disk usage — -sh for human readable size
- tar: archives files without compression by default — -z adds gzip compression
- gzip/bzip2/xz: compression tools with increasing ratio but decreasing speed — gunzip/bunzip2/unxz to decompress
- locate vs find: locate queries pre-built database (fast but may be outdated), find scans live filesystem
- stdin/stdout/stderr: three standard streams — keyboard input, screen output, screen errors
- 2> redirects stderr, > redirects stdout, /dev/null discards all output
- tee: sends output to screen AND file simultaneously
- grep -w: whole word match only, -v: exclude matches, -A/-B: show context lines around match
- vim three modes: command (default/Esc), insert (i,a,o,I,A,O), last line (:)
