## Learned
- uptime: shows system running time and CPU load averages over 1, 5 and 15 minutes
- type: identifies whether a command is built-in or external
- history: lists previously executed commands, reusable with !number
- pushd/popd: directory navigation with a stack — more powerful than cd - for complex navigation
- ls -ltr: standard way to find recently modified files — most recent at the bottom
- $PATH: list of directories where the shell searches for executable commands
- Built-in commands are integrated into the shell, external commands are programs on disk located via $PATH
