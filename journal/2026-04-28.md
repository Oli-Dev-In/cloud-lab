## Learned
- SSH: protocol for secure remote connection to any Linux server — local or remote. Requires SSH client (Windows) and SSH server (OpenSSH on Ubuntu). Same mechanism used to connect to AWS EC2 instances.
- Port forwarding: redirects connections from a host port to a guest port through VirtualBox NAT. Host 127.0.0.1:2222 → Guest 10.0.2.15:22.
- Linux filesystem structure: single tree starting from /. Everything is a file. Key directories: /home, /etc, /var, /usr, /bin, /tmp, /proc, /dev.
- Linux prompt: user@hostname:/path$ — $ means normal user, # means root.
- Navigation commands: pwd, ls, ls -l, ls -a, cd, cd .., cd ~, cd -, . and ..
- Essential commands: whoami, man, clear
- VirtualBox and SSH are distinct tools — VirtualBox creates the VM, SSH connects to it.
