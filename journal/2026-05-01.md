## Learned
- echo: prints text or variable value to terminal, can write to files with > and >>
- cat: displays full file content in terminal
- find: searches filesystem by filename, type, size or date
- grep: searches for a pattern inside file content — different from find which searches filenames
- wc: counts lines, words and bytes in a file
- >: redirects output to file, overwrites existing content
- >>: appends output to end of file, preserves existing content
- &: runs command in background, frees terminal immediately
- &&: runs second command only if first succeeded — used in scripts and CI/CD pipelines
