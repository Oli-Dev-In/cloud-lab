## Learned
- useradd vs adduser: useradd is low-level and scriptable, adduser is interactive — always use useradd in DevOps
- userdel -r: always use -r to clean up home directory
- /etc/passwd format: username:x:UID:GID:comment:home:shell
- /etc/group format: groupname:x:GID:members
- groupadd: creates a new group
- usermod -aG: adds user to group without removing from others — never use -G alone
- Pipe |: passes stdout of one command as stdin to the next — creates powerful command chains
- Package management families: Debian-based (dpkg/apt) vs RPM-based (rpm/yum) — not compatible
- APT vs DPKG: apt resolves dependencies automatically, dpkg does not
- Always verify after every system change: id, groups, grep /etc/passwd, grep /etc/group
