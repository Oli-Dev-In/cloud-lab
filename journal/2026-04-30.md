## Learned
- sudo: executes one command as root, then returns to normal user — principle of least privilege
- su: switches user identity for entire session until exit
- chmod symbolic: u/g/o/a + r/w/x with +/-/= operators
- chmod octal: r=4, w=2, x=1 — key values to memorize: 755, 644, 600, 700
- chown: changes owner and/or group, requires sudo
- id: shows UID, GID, all groups including sudo membership
- groups: shows group names only
