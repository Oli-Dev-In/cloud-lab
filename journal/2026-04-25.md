## Learned

- GitHub: the public-facing platform that hosts Git repositories — Git works locally, GitHub makes it visible and accessible
- Git three-zone workflow: Working Directory → Staging Area → Local Repository → GitHub
- Git commands: `git add` moves files to staging, `git commit` creates a local snapshot, `git push` transfers commits to GitHub
