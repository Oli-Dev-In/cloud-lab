# Journal

Daily learning log. One entry per session.

## Format

Each entry follows this structure:

- **Date** — session date
- **Focus** — what was studied or built
- **Done** — concrete tasks completed
- **Learned** — key concepts understood in my own words
- **Blocked** — problems encountered and how they were solved
- **Next** — what the next session will start with
