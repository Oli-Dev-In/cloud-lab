## Learned
- /sbin/nologin vs /bin/false — non-interactive shells for service accounts
- useradd -M — declares home directory in /etc/passwd but does not create it physically
- useradd -e — sets account expiry date, verified with chage -l not grep /etc/passwd
- find -type f -user — filter files by type and owner
- find -exec {} \; — execute command on each result, {} is placeholder, \; terminates
- cp --parents — preserves full directory structure during copy
- sshd_config vs ssh_config — server vs client configuration
- PermitRootLogin no — disables direct root SSH login
- Must restart service after config change: systemctl restart sshd
- systemctl status does not require sudo, systemctl restart does
